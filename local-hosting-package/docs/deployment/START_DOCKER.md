# 🐳 Start Docker Services

## Prerequisites
✅ Docker Desktop must be **RUNNING**

## Quick Start

```bash
cd local-hosting-package
docker compose up -d --build
```

## Access Links

- **Frontend**: http://localhost
- **Backend**: http://localhost:8002
- **API Docs**: http://localhost:8002/docs
